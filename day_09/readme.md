## Khai báo biến

Sử dụng từ khóa `let`, `const` để khai báo biến

### Sử dụng let

Khai báo biến bình thường

```javascript
// Khai báo biến
let number;

// Khai báo biến và khởi tạo giá trị
let age = 20;
```

### Sử dụng const

Được sử dụng để khai báo hằng số

```javascript
const number = 10
```

> Không thể gán lại giá trị cho 1 biến const
